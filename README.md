# CatchDogs
projeto da faculdade - facilitador de reencontros de animais perdidos


**Projeto**: CatchDogs — versão inicial  
**Feito por:** Pedro Almeida (Equipe do 1º semestre)  
**Tecnologias:** Java
**Ano/Semestre:** 1º semestre, 2025

## Descrição
CatchDogs foi o projeto desenvolvido no primeiro semestre da disciplina X. É um sistema simples para facilitar o reencontro de animais perdidos por meio da internet

## O que tem nesta versão
- Módulo: cadastro de animais, buscar por região, lista de animais perdidos etc...
- Banco de dados: persistência de dados atraves de .txt
- Interface: no terminal
